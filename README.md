# Satellite Physics

A simulation of planets orbitting a central mass

## Install and Run

**1. Create a virtual environment**

```sh
$ virtualenv venv
```

**2. Install dependencies**

```sh
$ pip3 install -r requirements.txt
```

**3. Start the project**

```sh
$ python3 start.py
```
